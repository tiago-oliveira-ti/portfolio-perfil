# Portfólio Pessoal — Tiago Oliveira

Página de apresentação pessoal construída com **HTML semântico**, **CSS puro** e **JavaScript vanilla**.  
Desenvolvida como primeiro projeto de portfólio de QA e desenvolvimento.

## 🖥️ Prévia
Abra o arquivo `index.html` diretamente no navegador ou publique nas **GitHub Pages**.

## 📁 Estrutura


 ```  projeto-1-perfil/
├── index.html        # Estrutura semântica da página
├── css/
│   └── style.css     # Estilos, variáveis CSS, responsividade
└── js/
└── main.js       # Animações, nav ativa, scroll suave


## ✅ Funcionalidades
- Design dark com tipografia refinada (DM Serif Display + DM Sans)  
- Cabeçalho fixo com pano de fundo desfocado e destaque da seção ativa  
- Animações de entrada suave com `IntersectionObserver`  
- Scroll suave com offset correto para o cabeçalho fixo  
- Layout totalmente responsivo (mobile e desktop)  
- Ano do rodapé atualizado automaticamente via JavaScript  
- Acessibilidade com atributos ARIA e semântica HTML5  

## 🛠️ Tecnologias
| Tecnologia       | Uso |

| CSS3             | Variáveis, Grid, Flexbox, animações |
| JavaScript ES6+  | Observador de Interseção, eventos |
| Google Fonts     | DM Serif Display + DM Sans     |
