# Portfólio Pessoal — Tiago Oliveira

Página de apresentação pessoal construída com **HTML semântico**, **CSS puro** e **JavaScript vanilla**. Desenvolvida como primeiro projeto do portfólio de QA e desenvolvimento.

## 🖥️ Preview

> Abrir o arquivo `index.html` diretamente no navegador ou subir no GitHub Pages.

## 📁 Estrutura

```
projeto-1-perfil/
├── index.html        # Estrutura semântica da página
├── css/
│   └── style.css     # Estilos, variáveis CSS, responsividade
└── js/
    └── main.js       # Animações, nav ativa, scroll suave
```

## ✅ Funcionalidades

- Design dark com tipografia refinada (DM Serif Display + DM Sans)
- Header fixo com backdrop blur e destaque da seção ativa
- Animações de entrada suaves com `IntersectionObserver`
- Scroll suave com offset correto para o header fixo
- Layout totalmente responsivo (mobile e desktop)
- Ano do footer atualizado automaticamente via JavaScript
- Acessibilidade com atributos ARIA e semântica HTML5

## 🛠️ Tecnologias

| Tecnologia | Uso |
|------------|-----|
| HTML5 | Estrutura semântica |
| CSS3 | Variáveis, Grid, Flexbox, animações |
| JavaScript ES6+ | Intersection Observer, eventos |
| Google Fonts | DM Serif Display + DM Sans |

## 🚀 Como rodar

1. Clone o repositório:

   git clone https://github.com/tiago-oliveira-ti/portfolio-nivel1.git
